Over the years, It has been proved that Linux is a very portable operative System capable of running on different hardware architectures. There is an important reference to portability in the discussion and this makes relevant the following statement. The Linux approach to portability includes architecture-independent interfaces and core code, presenting portable features that do not need much optimisation work (Love, 2010). Linux is also used a lot in cloud environments these days, allowing the implementation of microservices infrastructure to host software composed of independent services.

The fact that many organisations are adopting microservices architecture at the infrastructure and application level because of the ability to provide independent scalability and independent deployments does not prove that Linux is still a monolithic OS that will not be used in the future and is destinated to disappear as a monolithic OS solution.

Future Trends (Reading notes)

Blockchain Security

Blockchain security encloses encryption, distributed network and consensus to produce the structure of data with inherent security elements (IBM, N.D.).

It can maintain track of transactions by using interconnected blocks that represent a public key containing a history of transactions. The consensus concept refers to a security algorithm. To validate transactions according to the agreement of most nodes in the distributed network. To base the trust on this consensus means that there is no centralised trust authority.

An example of a blockchain platform is Bitcoin (cryptocurrency).

These platforms are not exempt from cyber attacks, as the number of incidents of this type can prove it.

Cyber-Physical Systems (CPS)

They are hybrid, meaning an intersection between both the cybernetic and physical world exists. These systems are also heterogeneous because they enclose different types of components. These opponents are distributed, meaning that they are separated physically and temporally.

CPSs are also Large-Scale systems, and their environment is dynamic. Dynamism enforces adaptability.

CPSs finally include the human factor, which is materialised with the interaction between devices and people.

An example of this type of system is Advanced driver assistance systems (ADAS).

References

IBM (N.D.) What is blockchain security?. Available from: https://www.ibm.com/topics/blockchain-security [Accessed 15th July 2023]
